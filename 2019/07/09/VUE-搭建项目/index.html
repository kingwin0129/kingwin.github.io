VUE 3.x项目搭建

自己看了很多遍VUE项目的各种搭建，但是时间久了就会忘记。所以特地从头到尾写一遍流程~ 好记性不如烂笔头！

准备环境

1.NodeJs 环境

官网下载链接: https://nodejs.org/en/download/

安装后查看版本，从而得知是否安装成功

如果出现对应的版本号，说明你已经成功了！






2.Vue-cli 环境

和node一样检查版本号，如果出现说明你已成功安装!






3.Vue-cli 创建项目

help命令：




















进入要创建项目的路径下运行命令

首先会让你选择你配置文件












运行命令后会出现对应的选择配置 分别为：

1.是否使用class风格的组件语法, 选择 yes






2.是否使用babel做转义， 选择 yes






3.是否使用路由的 history模式，选择 yes






4.选择css预处理器，请根据个人情况进行选择






5.选择 Eslint 的使用方式，我选择的 第二项






6.是否把以上的选择作为将来下个项目的模板进行保存，我选择的第一项 Lint on save.






7.单元测试，我选择使用 Jest






8.选择E2E测试解决方案，我选择使用 Nightwatch.






9.选择以上的配置文件放置的位置，一般选择第一项，单独配置。






10.是否保存以上的 preset 配置，可在将来创建项目的时候直接引用，我选择Yes.
